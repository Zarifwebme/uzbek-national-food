# O'zbek Milliy Taomlari (Uzbek National Food) 🇺🇿

O'zbekistonning milliy taomlari haqida ochiq ma'lumotlar bazasi va API loyihasi. Ushbu loyiha o'zbek pazandachilik merosini raqamlashtirish va dasturchilar uchun tayyor ma'lumotlar manbasini yaratish maqsadida tashkil etildi.

---

## Loyiha Maqsadi (Project Goal)
- Milliy taomlarimiz haqida aniq va tartibli ma'lumot yig'ish.
- Backend va Mobile dasturchilar uchun tekin va ochiq API (JSON) taqdim etish.
