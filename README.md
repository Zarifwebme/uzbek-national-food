# O'zbek Milliy Taomlari (Uzbek National Food) 🇺🇿

O'zbekistonning milliy taomlari haqida ochiq ma'lumotlar bazasi. Ushbu loyiha o'zbek pazandachilik merosini raqamlashtirish va dasturchilar uchun tayyor ma'lumotlar manbasini yaratish maqsadida tashkil etildi.

---

## Loyiha Maqsadi (Project Goal)
- Milliy taomlarimiz haqida ma'lumot yig'ish.
- Hamma o'zi bilgan taomni ro'yhatga qo'shishi mumkun
